# To-do
## Instruction

### Download To-do app
```sh
git clone https://github.com/HalaAsaad/To-do.git
```

### Installation
open a terminal inside To-do app, then:
```sh
npm install
```
to run in browser
```sh
npm start
```
you can build the project with a single command:
```sh
npm run build
```
## To-do Dependencies
* React:  is a JavaScript library for building user interfaces. 
* react-router-dom: is a standard library for routing in react.
* material-ui: UI component for React.
